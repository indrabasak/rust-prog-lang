## Two's Complement

Let's try find 2's complement of `5`.

  - Express `5` in binary form in 8-bits
   `00000101`
  
  - Invert the digits where 0 becomes 1 and 1 becomes 0
   `11111010`
   
  - Add 1 to `-5` in 2's complement
```
  11111010
 +       1
 ----------
  11111011
```
